## Usage

### Prerequisites
- Java Development Kit (JDK) installed on your system

### Steps to Compile and Run

```sh
# Clone the Repository
git clone https://github.com/hrishavvv/WaifuXJava
cd WaifuXJava

# Compile the Java File
javac WaifuHub.java

# Run the Compiled Java Program
java WaifuHub
